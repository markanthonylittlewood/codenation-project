Unit 2



1.1 - I used a simple structure for all my pages aside from the ‘projects’ page. Each page will have a central logo, a coloured navigation bar. Then each page will have an image and a footer. The projects page will have boxed images identifying my different, completed projects.

1.5 - I have ensured that all content and code included on my website is my own. Certain elements, such as the Navigation bar, are taken from the W3Schools website.

1.6 - Accessibility for people with disabilities is something which should be considered. For example, having larger for older visitors to the website or people with difficulty reading. A good choice of font is essential here as it will contribute to how readable the content is.

1.7 - I have used a number of file types in my website. I used an SVG for my logo. I chose this as it allowed me greater freedom to edit the image and also provides a crisper image. I used docx for my downloadable CV. For the main banner image, I used PNG. You could also use JPEG file format for images.
